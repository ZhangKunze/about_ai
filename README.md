##关于LLM现状

对于现在的LLM

我很害怕他会变成下一个资本的玩物，以及阻挡开源的东西。所以本仓库我要详细探讨一下这个事情。

首先来说一下我的观点:

#LLM(不是法人，也没有自主意识)→被一个合法的法人(公司、个人)的意志来做事

这一点我可以说我算是比较清醒的一类，我希望咱们对待任何事都要这样去发现它的本质，而不是被骗。就因为多了一层“AI不是法人，他没有自我意识”咱们就被骗了这么多年。也只能说是AI公司伪装的好，不然早就被发现了。

继续逻辑推理

#因为被合法法人的意志所受影响→LLM去做事→如果LLM训练的时候用了GPL(不管是几，或者衍生版)的代码，那么训练这个事情实际上就是约等于法人在进行学习，因为LLM训练这件事本身是法人的意志，所以就约等于法人侵害GPL的核心权益(自由)，所以你是可以直接起诉那个公司的。

###我自己也知道，如果我这个东西放在Github上肯定很快就会被下架的，道理也都懂，因为微软现在实际上就是个LLM公司，要是我这番言论影响了大众的话，那么它的核心利益就会受损，他为了避免利益受损，肯定就会先下架。

最后说一句话：开源现在越来越不好活了。为什么？因为LLM。这东西写出的代码就是短期正确，长期不正确。你根本没有办法分清哪个是人写的，哪个是LLM写的，导致了审核者的不便。

到最后了，我只能说：LLM这玩意儿信不得，用不得，使不得。千万不要为某个LLM公司而感到高兴，一旦你这么做了，就成了这场资本游戏的推进者。
